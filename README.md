# Personal Portfolio Website

This is the source code for my personal portfolio website, hosted on GitHub Pages.

## 🌐 Live Site

You can view the live website here: **[https://mahekb15.github.io/portfolio](https://mahekb15.github.io/portfolio)**

## 📌 Project Overview

This website serves as a central hub to showcase my professional profile, including my background in **Artificial Intelligence**, **Machine Learning**, and **Cyber Security**. It presents my education, technical skills, certifications, and key projects in a clean, modern format.

The primary goals of this project were to:
*   Create an online presence to display my work and skills.
*   Practice and demonstrate front-end web development using HTML and CSS.
*   Build a responsive and visually clean interface that works on different devices.

## 🛠️ Built With

*   **HTML5** - For the page structure and content.
*   **CSS3** - For all styling, layout (using Flexbox/Grid), and responsive design.
*   **Font Awesome** - For icons used throughout the site.
*   **Google Fonts** - For the 'Inter' typography.

## ✨ Features

*   **Clean & Modern UI**: A professional design with a light gradient background and subtle shadows.
*   **Fully Responsive**: Adapts seamlessly to desktops, tablets, and mobile phones.
*   **Project Showcase**: Dedicated cards to highlight key projects with descriptions.
*   **Skill Tags**: Interactive chips to visually list technical proficiencies.
*   **Contact Links**: Direct icons and links to GitHub, LinkedIn, and email.

## 📂 Project Structure

The website is a single-page application built from one main HTML file.
